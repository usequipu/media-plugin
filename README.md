# media-plugin

Image and video viewer plugin for [Quipu](https://quipu.cc). Displays images and videos with zoom (Ctrl+scroll).

## Install

Open the Plugin Manager panel in Quipu → Available → Install.

## Supported file types

Images: `.jpg`, `.jpeg`, `.png`, `.gif`, `.svg`, `.webp`, `.bmp`, `.ico`
Video: `.mp4`, `.webm`, `.ogg`, `.mov`

## Build from source

```bash
npm install
npm run build
```
